Vagrant recipe for development OpenStreetMap Drupal site
=====================================================

Preparation
--------

1. run ```./prepare.sh``` to get drupal files into files/drupal

2. Install vagrant

3. prepare `files/dbconfig.php` and `drupal-data.sql` for test data.

Getting started
--------

```vagrant up```
